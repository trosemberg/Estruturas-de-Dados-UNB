Apos abrir o terminal, na pasta Trabalho 1 siga os passos:
Compile o programa com o comando make.
Execute-o com o comando “./prog”. 
Em seguida, digite as entradas desejadas de acordo com o formato proposto pelo roteiro.
Para abrir os exemplos, execute-o como: 
./prog <input-base.txt		ou	./prog <data.txt
Limpe os arquivos gerado pelo comando “make” com o comando “make clean”.

Para gerar os arquivos utilizados no relatório, basta utilizar o programa geraExemplos.m
Para utilizá-lo, execute o programa com o comando “./prog <data.txt”.
